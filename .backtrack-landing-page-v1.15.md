## New Values (v1.16)

### src/app/page.tsx

**Hero H1:**
```tsx
<h1 className="text-[clamp(3rem,5vw,4rem)] font-bold leading-[1.1] tracking-[-0.03em] mb-8 text-[var(--color-text)]">
```
Changes:
- Size: 4.5rem → 4rem (max size reduced)
- Weight: 500 (font-medium) → 700 (font-bold)
- Removed inline style

**Hero Subtitle:**
```tsx
<p className="text-[1.25rem] text-[var(--color-text-body)] max-w-[600px] mb-12 leading-[1.6]">
```
Changes:
- Size: clamp(1.1rem,2vw,1.25rem) → 1.25rem (fixed)
- Color: --color-muted → --color-text-body
- Removed inline style, added explicit leading

**Footer H2:**
```tsx
<h2 className="text-[2rem] font-semibold mb-4 text-[var(--color-text)] tracking-[-0.01em] leading-[1.2]">
```
Changes:
- Size: clamp(2rem,3vw,2.5rem) → 2rem (fixed)
- Weight: font-medium → font-semibold (600)
- Added tracking and leading

**Footer Text:**
```tsx
<p className="text-[var(--color-text-body)] text-[1.25rem] leading-[1.6] mb-8 max-w-[65ch]">
```
Changes:
- Color: --color-muted → --color-text-body
- Size: text-base → 1.25rem (20px)
- Max-width: max-w-2xl → max-w-[65ch]
